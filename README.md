# 📝 ThinkBoard - Notes App

A full-stack **Notes Management Web Application** built using the MERN stack (React, Node.js, Express, MongoDB).

This application allows users to create, view, update, and delete notes with a clean and responsive UI.

---

## 🚀 Features

- 📝 Create new notes  
- 📖 View all notes  
- ✏️ Edit existing notes  
- ❌ Delete notes  
- ⚡ RESTful API integration  
- 📱 Responsive UI  

---

## 🔗 API Endpoints

| Method | Endpoint | Description |
|--------|----------|------------|
| GET    | /api/notes | Fetch all notes |
| POST   | /api/notes | Create new note |
| PUT    | /api/notes/:id | Update note |
| DELETE | /api/notes/:id | Delete note |
